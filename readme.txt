Trash or recycle menu items of menus

== Description ==
This plugin lets you trash menu items. So if you have a custom link in the menu that you want to delete but don't want to loose the link or title of the menu item, this plugin is for you. 

== Installation ==
1. Upload trash-menu-items to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Trash your Menu Items in Appearance -> Menus page

== How to use it ==

Once you activate the plugin, you will see the trash metabox on WP Admin > Appearance > Menus page.

1) To trash a menu item: expand any menu item by clicking on the arrow next to title of menu item.
2) Click on Trash button located next to cancel button.
3) Now you can see the trash menu items in trash meta box from where you can again add this items to menu or delete permanently.

== Frequently Asked Questions ==

= Trash Menu Item is broken =

The code for modifying the menu items is limited and if other plugins/themes try to alter the menu items, this plugin will break.

This is an ongoing [issue with WordPress](http://core.trac.wordpress.org/ticket/18584) which hopefully will be fixed in a future release.

Try to use just one plugin that changes functionality for menu items.
